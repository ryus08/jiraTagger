# jiraTagger
(modeled from [instructions by Muhammad Soliman](https://medium.com/hackernoon/go-serverless-offline-wait-go-yes-it-is-really-simplified-1dea663b97d5)

## CLI

### Start up the API Locally

```bash
    $ make api
```
### Unit Test 

```bash
    $ make test
```

### Deploy project remotely

```bash
    $ make deploy
```

### UnDeploy

```bash
    $ make undeploy
```
